Variables

JavaScript's variables are untyped. JavaScript uses lexical scoping meaning variables declared outside a function are global or visible to the entire program.

Numbers

JS does not make a distinction between integer values and floating-point or decimal values. JS represents numbers using the 64-bit floating point format defined by IEEE 754. It ranges from +/- 1.7976931348623157 * 10^308 to +/- 5 * 10^-324. Integers range between -9007199254740992 to 9007199254740992. Array indexing and bitwise operators are performed on 32-bit integers. Preceding a numeric literal with 0x will make it hexadecimal. Floating point literals may use e or E followed by an optional plus or minus sign followed by an integer exponent. This represents the number multiplied by 10 to the power of the exponent. For overflow, underflow, or division by zero the result is a special infinity value or NaN. NaN does not compare equal to any other value including itself.

Arithmetic

Arithmetic operators are '+', '-', '*', '/', '%'. The Math object also has a set of functions and constants for more complex operations.
